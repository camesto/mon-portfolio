Site Web de Démocratie Participative

Développement d’une plateforme permettant aux citoyens de proposer des idées et voter démocratiquement.

Ce projet vise à offrir un espace transparent et sécurisé où chacun peut s'exprimer librement et voir ses propositions être évaluées et votées.

Technologies utilisées :

  • HTML
  • CSS
  • PHP
  • MySQL
  • API REST

Fonctionnalités :

  • Inscription et connexion sécurisées
  • Création et gestion de groupes citoyens
  • Vote électronique sécurisé et anonyme
  • Notifications et suivi des décisions

Base de Données pour un Salon d'Esthétique

Conception et développement d'une base de données pour gérer efficacement les clients, rendez-vous et prestations d'un salon de beauté.

Le projet permet d'optimiser la gestion interne du salon et d'assurer un suivi client fluide.

Technologies utilisées :

  • SQL
  • MySQL

Fonctionnalités :

  • Gestion des clients et des réservations
  • Suivi des prestations et des paiements
  • Rapports analytiques sur les services les plus demandés

Application Java pour les Jeux Olympiques

Développement d'une application Java permettant de gérer les épreuves, les équipes et les performances des athlètes.

Le projet met en place une architecture bien structurée pour assurer une gestion fluide des données en lien avec les compétitions sportives.

Technologies utilisées :

  • Java
  • Swing

Fonctionnalités :

  • Gestion des athlètes et des équipes
  • Suivi des scores et des résultats en temps réel
  • Génération de statistiques et de classements
